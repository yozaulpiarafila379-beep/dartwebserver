1. Pengenalan Sealed Class

Sealed class adalah jenis class dalam Dart yang digunakan untuk membatasi pewarisan (inheritance), sehingga class tersebut hanya dapat diturunkan (di-extend) oleh class lain yang berada dalam file yang sama. Dengan adanya pembatasan ini, pengembang dapat mengontrol dengan jelas struktur turunan class, sehingga program menjadi lebih aman, terorganisir, dan mudah dipahami. Pada contoh di atas, ApiResponse digunakan sebagai sealed class yang memiliki dua turunan, yaitu Success dan Error. Kedua class tersebut merepresentasikan kondisi berhasil dan gagal, sehingga memudahkan dalam pengelolaan alur program dan pengecekan tipe data.

Kode Program VS Code:


// Nama  : Yoza Ulfia Rafila
// NIM   : 251410004
// Kelas : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void main() {
  print("Contoh penggunaan Sealed Class");
}

2. Keuntungan Penggunaan Sealed class

Keuntungan sealed class dalam Dart adalah dapat membatasi pewarisan hanya dalam satu file, sehingga semua subclass harus dibuat dalam file yang sama. Hal ini membuat program lebih terkontrol, aman, dan mudah dipahami karena semua kemungkinan turunan sudah diketahui. Pada kode di atas, ApiResponse digunakan sebagai sealed class dengan dua turunan, yaitu Success dan Error, yang mewakili kondisi berhasil dan gagal. Dengan adanya fungsi handleResponse(), program dapat melakukan pengecekan tipe untuk menampilkan hasil yang sesuai, sehingga alur program menjadi lebih rapi, jelas, dan terstruktur.

Kode Program VS Code:


// Nama  : Yoza Ulfia Rafila
// NIM   : 251410004
// Kelas : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void handleResponse(ApiResponse response) {
  if (response is Success) {
    print("Success: ${response.message}");
  } else if (response is Error) {
    print("Error: ${response.errorMessage}");
  }
}

void main() {
  handleResponse(Success("Data berhasil diambil"));
  handleResponse(Error("Terjadi kesalahan"));
}

3. Polimorfisme (Method Overriding)

Polimorfisme adalah konsep dalam pemrograman berorientasi objek yang memungkinkan sebuah method memiliki banyak bentuk perilaku. Salah satu bentuk polimorfisme adalah method overriding, yaitu ketika subclass mendefinisikan ulang method yang sudah ada pada superclass. Pada kode di atas, class Animal memiliki method speak() yang menampilkan suara umum. Kemudian class Dog sebagai turunan dari Animal melakukan overriding pada method tersebut untuk menghasilkan suara yang berbeda, yaitu "Woof!". Hal ini menunjukkan bahwa method yang sama dapat memiliki perilaku yang berbeda tergantung pada objek yang memanggilnya.

Kode Program VS Code:


// Nama  : Yoza Ulfia Rafila
// NIM   : 251410004
// Kelas : SI2A

class Animal {
  void speak() {
    print("Hewan bersuara");
  }
}

class Dog extends Animal {
  @override
  void speak() {
    print("Anjing: Woof!");
  }
}

void main() {
  Animal a = Animal();
  a.speak();

  Dog d = Dog();
  d.speak();
}

4. Method Overriding Pada Class Turunan

Method overriding adalah konsep dalam pemrograman berorientasi objek di mana class turunan (subclass) mendefinisikan ulang method yang sudah ada pada class induk (superclass). Tujuannya adalah untuk memberikan perilaku yang berbeda sesuai dengan kebutuhan masing-masing objek. Pada kode di atas, class Animal memiliki method speak() yang menampilkan suara umum. Kemudian class Dog dan Cat sebagai turunan melakukan overriding terhadap method tersebut untuk menghasilkan suara yang berbeda, yaitu "Woof!" dan "Meow!". Hal ini menunjukkan bahwa method yang sama dapat memiliki perilaku yang berbeda tergantung pada objek yang digunakan. Dengan demikian, method overriding membantu membuat program lebih fleksibel, mudah dikembangkan, dan mendukung konsep polimorfisme dalam pemrograman.

Kode Program VS Code:


// Nama  : Yoza Ulfia Rafila
// NIM   : 251410004
// Kelas : SI2A

class Animal {
  void speak() {
    print("Hewan bersuara");
  }
}

class Dog extends Animal {
  @override
  void speak() {
    print("Dog: Woof!");
  }
}

class Cat extends Animal {
  @override
  void speak() {
    print("Cat: Meow!");
  }
}

void main() {
  Animal a = Animal();
  a.speak();

  Animal d = Dog();
  d.speak();

  Animal c = Cat();
  c.speak();
}
