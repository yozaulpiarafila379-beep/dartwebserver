PENGERTIAN OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek sebagai unit utama. Setiap objek memiliki state (data) dan behavior (perilaku). Tujuan OOP adalah membuat kode yang modular, reusable, dan mudah dipelihara dalam sistem yang kompleks.

1. Class & Object (Car)

Contoh pendefinisian class Car dengan dua attribute (model dan color) serta dua method (start dan stop):

class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

Contoh pembuatan objek (instance) dari class Car:

void main() {
  Car myCar = Car();
  myCar.model = "Hatchback";
  myCar.color = "Biru";

  myCar.start();
  myCar.stop();
}
